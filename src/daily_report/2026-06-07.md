### 1. [CopilotKit/CopilotKit](https://github.com/CopilotKit/CopilotKit) · ⭐ 32,953 · TypeScript · 榜单第 3

面向 Agent 与生成式 UI 的前端栈，支持 React / Angular，背后是 AG-UI 协议的作者团队。把「给应用接一个会动手的 Agent」标准化成前端组件，是 Agent 应用落地的关键拼图。

要点：
- 生成式 UI（Generative UI）开箱即用，Agent 可直接驱动界面
- AG-UI 协议原生支持，定义 Agent 与前端的交互契约

### 2. [chopratejas/headroom](https://github.com/chopratejas/headroom) · ⭐ 15,182 · Python · 榜单第 2

在工具输出、日志、文件、RAG 切片到达 LLM 之前先压缩，号称减少 60–95% token。直击长上下文成本痛点，是 Agent / RAG 管线里立竿见影的省钱位。

### 3. [Panniantong/Agent-Reach](https://github.com/Panniantong/Agent-Reach) · ⭐ 21,942 · Python
